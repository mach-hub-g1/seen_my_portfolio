# Profile Picture Setup Instructions

## How to Add Your LinkedIn Profile Picture

### Method 1: Direct Download from LinkedIn
1. Go to your LinkedIn profile: https://www.linkedin.com/in/gajendra-singh-09688a348/
2. Right-click on your profile picture
3. Select "Save image as..." or "Copy image"
4. Save it as `profile.jpg` in the portfolio folder
5. Update the image source in `index.html`:
   ```html
   <img src="profile.jpg" alt="Gajendra Singh" id="profile-img">
   ```

### Method 2: Use the Interactive Feature
1. Open your portfolio website
2. Click on the profile image (it will show a camera icon on hover)
3. Select your profile picture file from your computer
4. The image will update automatically

### Method 3: Use LinkedIn Profile Picture URL
If your LinkedIn profile picture is public, you can use the direct URL:
1. Right-click on your LinkedIn profile picture
2. Select "Copy image address"
3. Replace the placeholder URL in `index.html` with your LinkedIn image URL
